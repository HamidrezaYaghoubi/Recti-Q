# Recti-Q paper website

The local paper website for **Recti-Q: Feature-Space Rectification for Out-of-Distribution-Robust Quantized Perception in Edge Robotics**.

## Local preview

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
```

The site is implemented with Next.js, React, and vinext. Paper-specific assets live in `public/`.
